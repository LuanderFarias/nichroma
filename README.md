~ nichroma [in development]

minimal game library manager & launcher compatible with multiple sources and some other fun stuff.

<img src="/.github/desktop.png"></img>
<img src="/.github/bigpicture.png"></img>
> nichroma is a spiritual successor of [cheep](https://github.com/LuanderFarias/Cheep)

nichroma is a simple to use and minimal game library manager & launcher that includes all your games in one place using a backend to automate your library management instead of spending hours dealing with configurations, downloading and managing many emulators, manualy removing uninstalled games, etc. made to be easy, local-first, customizable and fully extendable by the community. you can suggest functions, plugins and sources opening an [issue](https://github.com/LuanderFarias/nichroma/issues/new).
