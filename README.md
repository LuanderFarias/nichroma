~ nichroma

multifunctional program for computer utilities like cleanup, game modification, debloat and even some useless but fun stuff

<img src="example.gif"></img>

> nichroma is a spiritual successor of [cheep](https://github.com/LuanderFarias/Cheep)

~ modules
| module | description                |
| :-------- | :------------------------- |
| boost |  cleans the pc memory, stops unused background apps and adjusts a lot of other things to make your pc runs at the best performance possible. |
| lowspec | safely modify game files to make them run with custom performance by adjusting resolution, renderdistance, and more. |
| keyboard | test your type speed and make your keyboard sound as a mechanical keyboard. |
| keyoverlay | gives you a customizable keyboard and mouse overlay. |
| keypad | a module that lets you transform your cellphone or second monitor into a stream deck. |
| streaming | a fully featured screen recorder and stream software. |
| debloat (windows only) | gives you a collection of modifications you can apply to debloat your os. |
> nichroma is fully extendable so many modules will come in the future, you can also suggest modules opening an [issue](https://github.com/LuanderFarias/nichroma/issues/new)
